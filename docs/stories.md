# User Stories for Ocampo Apartments

## Users

- As a user, I want to register and log in with a unique username and email so
  that I can access the system securely.
- As a user, I want to have a role (Administrator, Building Admin, Tenant) so
  that my access and permissions are appropriate for my responsibilities.
- As an administrator, I want to activate or deactivate user accounts so that I
  can control access to the system.

## Tenants

- As a tenant, I want my personal and contact information (name, email, phone,
  Facebook) to be stored so that management can reach me easily.
- As a tenant, I want my account to be linked to my user login so that I can
  access tenant-specific features.

## Properties

- As an administrator, I want to add and manage properties with names and
  addresses so that all apartment units are organized by location.

## Apartment Units

- As an administrator, I want to add and manage apartment units with details
  (name, floor, price, capacity, status) so that I can track availability and
  assign tenants.
- As a tenant, I want to view available apartment units so that I can choose
  where to live.

## Tenancies

- As an administrator, I want to create and manage tenancies linking tenants to
  units with lease dates so that I can track who is renting which unit and for
  how long.
- As a tenant, I want to view my tenancy details so that I know my lease period
  and unit information.

## Bills & Bill Items

- As an administrator, I want to generate bills for each tenancy with due dates
  and status so that tenants know what they owe.
- As an administrator, I want to add multiple bill items (rent, utilities, etc.)
  to a bill so that charges are itemized.
- As a tenant, I want to view my bills and their breakdown so that I understand
  my charges.

## Payments

- As a tenant, I want to pay my bills using different methods (Over the counter,
  GCash) and upload proof so that my payments are recorded and verified.
- As an administrator, I want to track payments, including amount, date, and
  proof, so that I can confirm tenant payments and update bill status.

## Maintenance Workers

- As an administrator, I want to add and manage maintenance workers with contact
  details and availability so that I can assign them to requests.

## Maintenance Requests

- As a tenant, I want to submit maintenance requests for my unit, specifying
  urgency and description, so that issues are addressed promptly.
- As an administrator, I want to assign maintenance workers to requests and
  update their status so that requests are tracked from submission to
  completion.

## Announcements

- As an administrator, I want to create announcements with a title and message
  so that I can broadcast important information to all users.
- As a user, I want to view announcements so that I stay informed about updates
  and events.
